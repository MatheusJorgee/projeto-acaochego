# Documentação Técnica para Desenvolvedores

## Introdução
Este documento explica como o projeto funciona internamente e como configurá-lo para desenvolvimento.

## Estrutura de Pastas
- `/assets` — arquivos do frontend
- `/backend` — API e banco de dados
- `/docs` — documentação interna
- `/.github` — automações e templates de colaboração

## Requisitos
- Node.js v18 ou superior
- Banco de dados (MariaDB, MySQL ou SQLite)

## Como Rodar o Projeto
```bash
# Instalar dependências
npm install

# Rodar servidor backend
npm run dev
