# Arquitetura do Projeto

## Visão Geral
O projeto é dividido em três camadas principais:

1. **Frontend (assets/)**  
   - HTML/CSS/JS  
   - Renderização visual e interação com usuário

2. **Backend (backend/)**  
   - API REST
   - Comunicação com banco de dados

3. **Database (database/)**  
   - Tabelas
   - Schemas
   - Migrations futuras

---

## Diagrama Simplificado
Frontend -> Backend -> Database
